Deutsch:

Einsatz_Ticker
Ticker Für Einsatze in NÖ. in Ö 

Benötigt:
Pyaton: 3.11.3
https://www.python.org/downloads/release/python-3113/

Beschreibung:
Der obige Code ist ein Python-Skript, das einen Einsatz-Ticker für Feuerwehreinsätze in Niederösterreich darstellt. Es ermöglicht dem Benutzer, einen Bezirk in Niederösterreich auszuwählen und Informationen über laufende Einsätze oder Einsatz-Rückblicke zu erhalten. 
Das Skript verwendet mehrere Module, um diese Informationen abzurufen und anzuzeigen, und führt zu Beginn des Skripts einige Tests durch, um sicherzustellen, dass alle benötigten Module installiert sindund eine Internetverbindung besteht.
 Zuerst werden die benötigten Module überprüft und bei Bedarf installiert. Anschließend wird die Internetverbindung getestet, um sicherzustellen, dass alle erforderlichen Verbindungen hergestellt werden können. Dann kann der Benutzer ein Bundesland (in diesem Fall Niederösterreich) auswählen, und das Skript fordert den Benutzer auf, einen Bezirk und den gewünschten Bereich (laufende Einsätze oder Einsatz-Rückblicke) auszuwählen.
dann wird die "Start" Funktion aufgerufen, um die entsprechenden Informationen abzurufen und anzuzeigen. Die Informationen werden in einer Tabelle angezeigt, wobei verschiedene Farben verwendet werden, um die Art des Einsatzes (Brand, Technischer Einsatz, Sonstiges oder Dienstleistung) darzustellen.
 Wenn der Benutzer die Option "Exit" wählt, wird das Skript beendet.

English:

Operation_Ticker
Ticker for operations in Lower Austria (NÖ), in Austria

Requirements:
Python: 3.11.3
https://www.python.org/downloads/release/python-3113/

Description:
The above code is a Python script that displays an operation ticker for fire department operations in Lower Austria. It allows the user to select a district in Lower Austria and get information about ongoing operations or operation reviews.
The script uses several modules to retrieve and display this information and conducts several tests at the beginning of the script to ensure that all required modules are installed and an internet connection is available.
First, the required modules are checked and installed if necessary. Then, the internet connection is tested to ensure that all required connections can be established. Then, the user can select a federal state (in this case, Lower Austria), and the script prompts the user to select a district and the desired area (ongoing operations or operation reviews).
Then, the "Start" function is called to retrieve and display the corresponding information. The information is displayed in a table, using different colors to represent the type of operation (fire, technical operation, other, or service).
If the user selects the "Exit" option, the script will end.
